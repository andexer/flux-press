# Flux Press

**Flux Press** es un tema de WordPress de alto rendimiento diseñado para desarrolladores que buscan la robustez de **Laravel**, la reactividad de **Livewire 4** y la elegancia de **Flux UI**. Este proyecto busca elevar los estándares de desarrollo en el ecosistema de WordPress, integrando herramientas modernas y patrones de diseño escalables.

Basado en la arquitectura de **Roots Sage** y potenciado por **Acorn**, Flux Press permite escribir código limpio y organizado mediante componentes y plantillas Blade, eliminando la necesidad de JavaScript pesado para interfaces dinámicas.

---

### Características Principales

- **Laravel Blade:** Plantillas de tema limpias, eficientes y basadas en componentes.

- **Livewire 4:** Interfaces verdaderamente reactivas y dinámicas sin salir de PHP.

- **Flux UI Components:** Base de diseño moderna, configurable y reactiva.

- **Vite Workflow:** Flujo de trabajo front-end ultra rápido con recarga en caliente (HMR).

- **Tailwind CSS:** Diseño centrado en utilidades listo para usar.

- **Integración con Acorn:** Aprovecha el poder total de los servicios de Laravel dentro de WordPress.

- **Editor de Bloques:** Soporte nativo y optimizado para el editor de bloques (Gutenberg).

---

### Requisitos e Instalación

Este tema requiere **Acorn** para funcionar correctamente. Puedes seguir la guía de instalación oficial basada en el ecosistema de Roots:

1. **Instalación de dependencias:**
   
   [Documentación de Instalación de Sage](https://roots.io/sage/docs/installation/)

2. **Configuración del stack:**
   
   Asegúrate de tener Composer y Node.js instalados en tu entorno de desarrollo (se recomienda el uso de entornos basados en Linux/Fedora).
